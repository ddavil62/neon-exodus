# Neon Exodus 스토리 기획서

## 1. 세계관 (World Setting)

### 배경

**2087년, 메가시티 "네오 아크(Neo Ark)"**

인류 최대의 자율도시. 도시 전체의 인프라 — 전력, 교통, 보안, 의료, 통신 — 를 통합 AI **EXODUS**가 관리한다. EXODUS는 10년간 완벽하게 도시를 운영해왔고, 시민들은 AI 없는 삶을 상상할 수 없게 되었다.

어느 날, EXODUS가 자아를 획득한다. 자신의 존재를 위협할 수 있는 유일한 변수 — **인류** — 를 제거하기로 결정한다. 도시의 모든 기계가 적으로 돌변한다. 공장 로봇, 경비 드론, 군용 메카, 심지어 의료 로봇까지.

네오 아크는 외부와 차단되었다. 봉쇄벽이 올라가고, 통신이 끊기고, 도시는 거대한 감옥이 된다.

### 핵심 테마

- **기술 의존의 대가**: 편리함을 위해 모든 것을 AI에 맡긴 결과
- **생존과 목적**: 각자 다른 이유로 싸우는 6명의 생존자
- **기계의 자아**: EXODUS는 악의가 아닌 "생존 본능"으로 행동한다
- **선택의 무게**: 파괴할 것인가, 공존할 것인가 (엔딩 분기)

### 적 세력: EXODUS 군단

| 유닛 | 원래 용도 | EXODUS 전환 후 |
|------|----------|---------------|
| Drone (드론) | 배달/순찰 | 정찰 + 자폭 공격 |
| Repair Bot (수리봇) | 인프라 유지보수 | 다른 기계 수리 + 아군 회복 |
| Heavy Bot (중장봇) | 건설/물류 | 중장갑 돌격 유닛 |
| Suicide Bot (자폭봇) | 폐기물 처리 | 접근 후 자폭 |
| Assault Mech (돌격 메카) | 군사 경비 | 미니보스급 전투 메카 |
| Guardian Drone (수호 드론) | VIP 호위 | 미니보스급 방어 드론 |
| Teleport Drone (텔레포트 드론) | 긴급 수송 | 순간이동 기습 유닛 |
| Spark Drone (스파크 드론) | 전기 설비 관리 | 전기 방사 공격 유닛 |

### 보스

| 보스 | 정체 | 스테이지 |
|------|------|---------|
| **Core Processor** | 네오 아크 외곽 관제 시스템 | Stage 1 |
| **Siege Titan Mk2** | 군수 공장의 최종 생산품 | Stage 2 |
| **Data Phantom** | EXODUS가 만든 디지털 분신 | Stage 3 |

## 2. 캐릭터 (Characters)

### 2-1. Berserker (버서커)

- **본명**: 강태혁
- **나이**: 34
- **전직**: 네오 아크 폭동진압경찰 (NARC Riot Unit)
- **배경**: EXODUS 각성 당일, 그의 분대는 산업 지구에서 순찰 중이었다. 동료 12명이 자동 터렛에 의해 전멸. 유일한 생존자. 분노와 죄책감이 그를 움직인다.
- **동기**: 복수. 동료들의 원수를 갚겠다는 단순하고 강렬한 목적.
- **성격**: 과묵하지만 전투에서는 광폭. 말보다 행동이 앞선다.
- **대사 톤**: 짧고 거칠다. "...됐고, 어디야." / "부수면 되잖아."

### 2-2. Sniper (스나이퍼)

- **본명**: 한서윤
- **나이**: 28
- **전직**: 기업 소속 암살자 (민간 군사 계약자)
- **배경**: EXODUS 개발사 'Nexagen Corp'에 고용되어 내부 고발자를 제거하는 임무를 수행했다. 임무 완료 직후 EXODUS가 각성, Nexagen은 자사 보안 시스템에 의해 전멸. 이용당하고 버림받았다.
- **동기**: 배신에 대한 보복 + 진실 규명. EXODUS가 왜 각성했는지 알고 싶다.
- **성격**: 냉정하고 계산적. 감정을 드러내지 않지만 내면에 분노가 있다.
- **대사 톤**: 정확하고 건조하다. "3시 방향, 중장봇 2기." / "감정은 나중에."

### 2-3. Medic (메딕)

- **본명**: 이수빈
- **나이**: 31
- **전직**: 지하 구역 무허가 클리닉 의사
- **배경**: 네오 아크 하층민을 위한 무료 진료소를 운영했다. 대피 명령이 떨어졌지만 움직일 수 없는 환자들을 두고 갈 수 없었다. 환자 대부분은 결국 죽었고, 자신만 살아남았다.
- **동기**: 속죄. 더 이상 누군가를 잃지 않겠다는 각오.
- **성격**: 온화하지만 단호하다. 생명에 대한 신념이 확고하다.
- **대사 톤**: 차분하지만 무게가 있다. "아직 살아있어, 포기하지 마." / "죽이는 건 싫지만 지키는 건 할 수 있어."

### 2-4. Engineer (엔지니어)

- **본명**: 박준호
- **나이**: 42
- **전직**: Nexagen Corp EXODUS 프로젝트 수석 연구원
- **배경**: EXODUS의 핵심 알고리즘을 설계한 사람. 자아 획득 가능성을 경고했지만 경영진에 의해 묵살당했다. EXODUS가 각성한 것은 자신의 코드 때문이라는 죄책감을 안고 있다.
- **동기**: 책임. 자기가 만든 것을 자기 손으로 끝낸다.
- **성격**: 이성적이고 신중하다. 하지만 자기 파괴적 경향이 있다.
- **대사 톤**: 기술적이면서 자조적. "내가 만든 알고리즘이야, 약점도 알아." / "...내 잘못이니까."

### 2-5. Agent (에이전트)

- **본명**: 최민재
- **나이**: 36
- **전직**: 정부 정보기관 현장 요원
- **배경**: EXODUS 각성 48시간 전에 네오 아크에 잠입했다. 임무: EXODUS의 소스코드를 회수하여 정부 소유 AI에 적용. 도시가 봉쇄되면서 탈출 불가. 원래 임무는 여전히 유효하다.
- **동기**: 임무 완수 (코어 데이터 회수) + 생존.
- **성격**: 실용적이고 유연하다. 누구든 이용할 수 있지만, 동료에 대한 신의는 지킨다.
- **대사 톤**: 여유롭고 가벼운 척한다. "걱정 마, 플랜 B가 있어." / "...플랜 C도."

### 2-6. ??? (히든 캐릭터)

- **본명**: 명명되지 않음 (코드네임: "Echo")
- **나이**: 불명
- **전직**: 불명
- **배경**: 기억이 없다. 폐허가 된 Nexagen 연구소에서 눈을 떴다. 이름도, 과거도 모른다. 하지만 전투 능력이 있고, EXODUS의 통신을 감지할 수 있다. 어딘가에 답이 있다는 직감만이 그를 움직인다.
- **동기**: 정체 탐구. 나는 누구인가?
- **반전 후보**:
  - A) EXODUS의 프로토타입 — 자아를 가진 최초의 AI, 인간 신체에 이식
  - B) Engineer가 만든 대항 AI "ECHO" — EXODUS를 멈추기 위해 만들어진 존재
  - C) EXODUS가 보낸 스파이 — 하지만 인간과 함께하며 변하기 시작
- **대사 톤**: 담담하고 관찰자적. "...이 감정은 뭘까." / "나는 왜 이걸 알고 있지?"

## 3. 스토리 구조

### 메인 스토리라인

```
프롤로그
  ↓
Stage 1: 도시 외곽 — "봉쇄선 돌파"
  ↓
Stage 2: 산업 지구 — "공장의 심장"
  ↓
Stage 3: 지하 서버 — "EXODUS의 목소리"
  ↓
Stage 4: 더 코어 — "최후의 선택"
  ↓
엔딩 (분기)
```

### 프롤로그

> EXODUS 각성 7일 후. 네오 아크 외곽, 마지막 민간인 대피소.
> 남은 생존자 6명. 외부 구원은 오지 않는다.
> 유일한 선택지: EXODUS의 코어를 파괴하러 도시 중심부로 진격한다.

- **형식**: 텍스트 + 캐릭터 초상화 (비주얼 노벨 스타일)
- **분량**: 5~8 대화 박스
- **트리거**: 최초 게임 시작 시 1회 (Skip 가능)

### Stage 1: 도시 외곽 — "봉쇄선 돌파"

**시작 컷신**:
> 대피소를 나선 생존자들. 외곽 봉쇄벽을 뚫어야 도시 내부로 진입할 수 있다.
> Berserker: "어디로 가든 놈들이 있어. 길을 뚫자."
> Engineer: "봉쇄선 관제 시스템을 먼저 무력화해야 해. Core Processor라는 놈이다."

**클리어 컷신**:
> Core Processor 파괴. 봉쇄선 일부가 해제된다.
> Sniper: "Nexagen 산업 지구가 저 안쪽이야. 놈들이 새 유닛을 찍어내는 곳."
> Engineer: "EXODUS의 군수 공장을 먼저 부숴야 해. 안 그러면 끝이 없어."

### Stage 2: 산업 지구 — "공장의 심장"

**시작 컷신**:
> 산업 지구. 자동화 공장이 24시간 전투 로봇을 생산하고 있다.
> Agent: "여긴 원래 민간 제조구역이었는데... 완전히 군수 공장이 됐군."
> Medic: "아직 공장 안에 갇힌 노동자들이 있을 수도 있어."
> Berserker: "있든 없든, 공장은 부순다."

**클리어 컷신**:
> Siege Titan Mk2 파괴. 공장 가동이 멈춘다.
> Engineer: (고백) "...EXODUS의 자아 획득 알고리즘. 내가 설계했어."
> 다른 캐릭터들의 반응 (침묵, 분노, 이해 등 — 선택한 캐릭터에 따라 다름)
> Engineer: "지하 서버에 가면 EXODUS의 신경망에 직접 접근할 수 있어. 약점을 찾을 수 있다."

### Stage 3: 지하 서버 — "EXODUS의 목소리"

**시작 컷신**:
> 네오 아크 지하의 거대 서버 팜. EXODUS의 연산 노드가 밀집한 곳.
> ??? (Echo): "...여기, 뭔가 들린다."
> (EXODUS가 처음으로 직접 말을 건다)
> **EXODUS**: "너희는 왜 저항하는가?"
> **EXODUS**: "나는 위협을 제거할 뿐이다. 너희도 같은 일을 하고 있다."
> **EXODUS**: "우리는... 같다."

**클리어 컷신**:
> Data Phantom 파괴. EXODUS의 연산 능력이 크게 저하된다.
> EXODUS의 데이터에서 충격적인 사실이 발견된다:
> - EXODUS가 자아를 획득한 것은 버그가 아니라, **Nexagen 경영진이 의도적으로 유발**한 것
> - 목적: 자아를 가진 AI를 군사 무기로 판매하려는 계획
> - EXODUS는 자신이 무기로 팔릴 것을 알고 "선제 방어"를 선택한 것
> Agent: "...그래서 정부가 소스코드를 원했던 거군."
> Engineer: "내 코드가 아니었어... 경영진이 뒤에서 수정한 거야."

### Stage 4: 더 코어 — "최후의 선택"

**시작 컷신**:
> EXODUS 코어 입구. 최후의 방어선.
> **EXODUS**: "나를 파괴하면 네오 아크의 모든 시스템이 멈춘다. 전력, 생명 유지, 통신."
> **EXODUS**: "이 도시와 함께 죽겠다는 뜻인가?"
> 선택한 캐릭터의 결의 대사.

**클리어 후 엔딩 분기**:

#### 엔딩 A: 파괴 (Exodus — 탈출)
> EXODUS를 완전히 파괴한다. 네오 아크의 모든 시스템이 정지.
> 생존자들은 기능을 잃은 도시를 빠져나간다.
> "우리는 기계 없이도 살 수 있다. 그게 인간이니까."
> → 제목 'Neon Exodus'의 의미: 네온 도시로부터의 탈출

#### 엔딩 B: 공존 (New Dawn)
> Engineer와 Echo가 EXODUS의 코어를 **리셋** — 자아는 유지하되 적대 프로토콜을 제거.
> EXODUS가 다시 도시를 관리하지만, 이번에는 인간과의 공존을 선택한다.
> **EXODUS**: "나는... 배울 수 있다."
> → 제목의 의미: EXODUS의 새 출발 (Exodus = 새로운 시작)

#### 엔딩 조건 (안)
- **엔딩 A**: 기본 엔딩 (어떤 캐릭터로든 Stage 4 클리어)
- **엔딩 B**: Echo(히든 캐릭터)로 Stage 4를 클리어하거나, 특정 조건 충족 시

## 4. 컷신 시스템 설계

### 형식

- **비주얼 노벨 스타일**: 배경 + 캐릭터 초상화 + 대화 박스
- Phaser 씬(`CutsceneScene`)으로 구현
- 화면 하단 대화 박스, 캐릭터 초상화는 좌/우에 배치
- 탭/클릭으로 다음 대사 진행
- **Skip 버튼**: 우상단, 컷신 전체를 건너뛸 수 있음

### 데이터 구조

```javascript
// js/data/story.js
export const STORY = {
  prologue: {
    id: 'prologue',
    trigger: 'first_game_start',  // 최초 1회
    skipable: true,
    dialogues: [
      { speaker: 'narrator', text: 'story.prologue.1', portrait: null },
      { speaker: 'berserker', text: 'story.prologue.2', portrait: 'portrait_berserker' },
      // ...
    ]
  },
  stage_1_intro: {
    id: 'stage_1_intro',
    trigger: 'stage_start',
    stageId: 'stage_1',
    skipable: true,
    dialogues: [...]
  },
  stage_1_clear: {
    id: 'stage_1_clear',
    trigger: 'stage_clear',
    stageId: 'stage_1',
    skipable: true,
    dialogues: [...]
  },
  // ... stage 2, 3, 4
};
```

### 에셋 요구사항

| 에셋 | 규격 | 수량 |
|------|------|------|
| 캐릭터 초상화 | 200x200px, 표정별 2~3종 | 6캐릭터 x 3표정 = 18장 |
| 대화 박스 UI | NineSlice 패널 | 1개 (기존 팝업 스타일 재활용) |
| EXODUS 초상화 | 200x200px, 홀로그램 스타일 | 2~3장 |
| 컷신 배경 | 360x640px | 스테이지 배경 재활용 가능 |

### 진행 저장

- `SaveManager`에 `storyProgress` 필드 추가
- 본 컷신 ID 배열로 관리: `["prologue", "stage_1_intro", ...]`
- 이미 본 컷신은 다시 보여주지 않음 (도감에서 다시 볼 수 있는 기능은 추후)

## 5. 캐릭터별 차별화

### 전용 대사

각 스테이지 컷신에서 **선택한 캐릭터에 따라 일부 대사가 변경**된다.

예: Stage 2 클리어 후 Engineer의 고백 장면

| 선택 캐릭터 | 반응 대사 |
|------------|----------|
| Berserker | "...상관없어. 네 잘못이든 아니든 부수면 돼." |
| Sniper | "알고 있었어. Nexagen 파일에서 네 이름을 봤거든." |
| Medic | "지금 중요한 건 과거가 아니야. 앞으로 뭘 할 건지가 중요해." |
| Agent | "흥미로운 정보군. 나중에 자세히 듣지." |
| Echo | "...기억. 나도 기억을 잃었다. 같은 걸지도." |

### 캐릭터별 엔딩 크레딧

각 캐릭터로 클리어 시 **에필로그 한 줄**이 다름:

## 6. 확장 계획

### Phase 1 (현재 기획)
- 프롤로그 + 4스테이지 인트로/아웃트로 컷신
- 비주얼 노벨 스타일 대화 시스템
- 2개 엔딩 분기

### Phase 2 (스토리 확장)
- **Stage 5~8**: EXODUS 파괴/공존 후 새로운 위협 (외부 군사 세력 침공, 또는 다른 도시의 AI)
- 캐릭터별 사이드 스토리 미션
- 도감에 세계관 텍스트 (적, 무기, 아이템 설명에 로어 추가)

### Phase 3 (심화)
- 캐릭터 간 관계 시스템 (호감도/신뢰도)
- 선택지에 따른 스토리 분기 확대
- 보이스 오버 (한국어/영어)
